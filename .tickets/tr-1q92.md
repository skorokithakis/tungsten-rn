# Guard against empty screen titles in CarDataStore

In CarDataStore.kt:parseScreensJson(), the title defaults to empty string via optString('title', ''). Empty titles can crash Row.setTitle() and ListTemplate.setTitle(). Default to 'Untitled' instead of empty string.
